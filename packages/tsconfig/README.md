# @statter/tsconfig

Reusable TypeScript config presets for Node.js and React projects.

## Install

```bash
yarn add -D @statter/tsconfig typescript
```

## Usage

Node (`tsconfig.json`):

```json
{
  "extends": "@statter/tsconfig/tsconfig.node.json"
}
```

React (`tsconfig.json`):

```json
{
  "extends": "@statter/tsconfig/tsconfig.react.json"
}
```
