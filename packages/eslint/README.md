# @statter/eslint

Reusable ESLint flat config for TypeScript projects with Prettier integration.

## Install

```bash
yarn add -D @statter/eslint eslint
```

## Usage

Create `eslint.config.mjs`:

```js
import config from "@statter/eslint"

export default config
```

## Build (for maintainers)

```bash
yarn build
```
